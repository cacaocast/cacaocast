Épisode 95 - Cocoaheads, Tokens, INAppStoreWindow, POP, Objective-cloud.com, TransformerKit, ObjcLibs, Debug

21 novembre 2012

Bienvenue dans le quatre-vingt-quinzième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Cocoaheads Montréal - Nouveau site
  • Cocoaheads TV - Un seul arrêt pour les vidéos Cocoaheads
  • Tokens - Distribuer des codes promo est un jeu d’enfant
  • INAppStoreWindow - Le look “App Store” pour vos fenêtres
  • Marketing - Comment créer une vidéo pour votre app
  • POP - Prototypage sur papier et iOS
  • Objective-cloud.com - Objective-C dans le nuage
  • TransformerKit - Modernisation d’un vieil ami
  • ObjcLibs - Un supermarché pour les frameworks?
  • Debug - Un nouveau podcast pour les développeurs
