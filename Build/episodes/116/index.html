Épisode 116 - Tags dans Mavericks, MobileProvision, powermetrics, documentation Xcode5, MRProgress, My dad reads Xcode errors

31 octobre 2013

Bienvenue dans le cent-seizième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Les Tags dans Mavericks - Pour “tagger” en ligne de commande. L'autre version ici.
  • MobileProvision - QLMobileProvision fait un retour sous Mavericks
  • powermetrics - Un nouvel outil sous Mavericks pour diagnostiquer les applications gourmandes en énergie
  • Xcode5 Documentation - Un bon résumé des nouvelles fonctions
  • MRProgress - Indicateurs de progression dans le style iOS 7
  • My dad reads Xcode errors - Très drôle!
