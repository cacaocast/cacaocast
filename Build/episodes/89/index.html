Épisode 89 - Cupertino, NSBundle, PonyDebugger, tops, C et GDB

07 septembre 2012

Bienvenue dans le quatre-vingt-neuvième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • cupertino - Accès au portail de provisionnement par la ligne de commande
  • NSBundle -appStoreReceiptURL - La petite astuce du jour sous 10.7 (via Gus Mueller)
  • PonyDebugger - Pour “débugger” vos applications iOS à distance
  • tops - Un outil méconnu en ligne de commande
  • C et GDB - Apprendre le langage C avec GDB
  • “Unrecognized selector sent to instance” - Les breakpoints symboliques pour les nuls (merci à Simon Liotier)
