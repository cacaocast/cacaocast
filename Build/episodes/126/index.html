Épisode 126 - Kimono, ui-screen-shooter, Tweaks, TunableSpec, DB5, Keyboard Layout Guide

26 mars 2014

Bienvenue dans le cent-vingt-sixième épisode de CacaoCast! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Kimono - Pour extraire les données d'un site web
  • ui-screen-shooter - Mise-à-jour pour Xcode 5.1
  • Tweaks - Pour paufiner vos applications iOS en développement
  • TunableSpec - Comme Tweaks mais en deux fichiers seulement
  • DB5 - Les utilitaires de VesperApp
  • Keyboard Layout Guide - Pour conserver vos UITextFields à l’écran avec Autolayout
