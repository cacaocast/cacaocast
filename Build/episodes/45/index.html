Épisode 45 - OS X Lion?, GitX, HTTP Fetcher et Mobile Bookmark Bubble

14 octobre 2010

Bienvenue dans le quarante-cinquième épisode de CocoaCast Cacao! Dans cet épisode,Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • JBNHexView - Philippe a oublié de mettre JBNHexView dans les notes de l'épisode 44
  • OS X Lion? - Mac OS X est de retour!
  • GitX - Une nouvelle branche pour ce projet qui languit
  • HTTP Fetcher - Une classe pour récupérer des données HTTP
  • Mobile Bookmark Bubble - Petite librarie pour indiquer aux usagers comment mettre votre appli web sur leur iPhone
