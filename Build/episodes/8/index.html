Épisode 8 - Class Dump, KNAppGuide, JRFeedbackProvider, Quiz sur le langage C et Cappuccino 0.7

20 mai 2009

Bienvenue dans le huitième épisode de CocoaCast Cacao! Dans cet épisode, Philippe Casgrain et Philippe Guitard discutent des sujets suivants:

  • Class Dump - Présente les informations obtenues par otool de façon plus claire
  • KNAppGuide - Un framework pour ajouter des “guides” à vos applications
  • JRFeedbackProvider - Obtenez des informations de la part de vos utilisateurs
  • Quiz sur le langage C et Objective-C - Testez vos connaissances du language C et Objective-C
  • Cappuccino 0.7 - Tout nouveau!
